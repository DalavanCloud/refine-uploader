Data Upload
Server URL:
Bulk uploads an array of JSON objects representing your Refine project rows to a remote server via HTTP POST. Your server will have to know how to accept, parse and store JSON in the following format:
{
  "docs": [
    {"name": "barry", "type": "human"}, // first row
    {"name": "carl",  "type": "human"}, // second row
    {"name": "ethel", "type": "robot"}  // third row, etc...
  ]
}
